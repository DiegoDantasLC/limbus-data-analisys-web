Análise de IDs de Limbus Company

Projeto de Visualização e Análise de Dados

Introdução

Este projeto tem como objetivo analisar as IDs presentes em Limbus Company utilizando Python, Pandas, Google Colab, ObservableHQ e Vega-Lite.

Os dados foram organizados em formato CSV e posteriormente processados para gerar visualizações que permitem identificar padrões relacionados às seasons, raridades e distribuição de personagens.

Descrição do Dataset

Coluna Descrição
id_name Nome da ID.
sinner Personagem associado à ID.
rarity Raridade da ID (0, 00 ou 000).
season Season em que a ID foi lançada.

Metodologia

Inicialmente os dados foram coletados e organizados em uma planilha CSV.

Em seguida, utilizando a biblioteca Pandas, foram realizadas etapas de leitura, limpeza e análise dos dados.

Por fim, os dados foram exportados para o ObservableHQ, onde foram criadas visualizações utilizando Vega-Lite.
